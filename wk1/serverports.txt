1.What is the definition of server port?
--------------------------------------
A server port is a software construct that identifies a specific process to which network traffic is directed.
Ports provide a way for multiple applications on a single device to share a network connection.
They are identified by a number, known as the port number, which is included in the packet's header to ensure that the
data reaches the correct application or service on the destination device.

2.What is server port range?
-------------------------
The standard range of port numbers is from 0 to 65535

4. State 10 common server ports
------------------------------
Port 20: FTP data transfer

Port 25: Simple Mail Transfer Protocol (SMTP)
Port 53: Domain Name System (DNS) service
Port 80: Hypertext Transfer Protocol (HTTP) used in the World Wide Web

Port 119: Network News Transfer Protocol (NNTP)
